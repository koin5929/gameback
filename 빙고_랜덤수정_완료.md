# ✅ 빙고판 랜덤 + 보상 숨김 수정 완료

## 🎯 수정 사항

### 1. 빙고 아이템 랜덤 섞기 ✅
- 위치가 매번 랜덤하게 섞임
- 순서대로 정렬되지 않음

### 2. 보상 숨김 ✅
- 선택하지 않은 칸: "?" 표시
- 선택한 칸만: 보상 정보 표시

### 3. 클릭 이벤트 개선 ✅
- 포인트가 3 이상일 때만 클릭 가능
- 호버 시 확대 효과

---

## 🎮 동작 방식

1. **초기 상태**: 모든 칸이 "?"
2. **클릭 시**: 포인트 3개 차감
3. **선택 후**: 그 칸만 보상 정보 드러남
4. **위치**: 매번 새로고침할 때마다 랜덤
