# ✅ 포인트 비율 변경 완료

## 📊 변경된 포인트 시스템

### 획득
- **30분 플레이 = 1포인트**

### 소모 (변경됨)
- **빙고 칸 1개 선택 = 3포인트** (기존: 1포인트)

## 📈 계산

### 예시 1:
```
3시간 플레이
→ 180분 ÷ 30분 = 6포인트
→ 6포인트 ÷ 3 = 2칸 선택 가능
```

### 예시 2:
```
1시간 플레이
→ 60분 ÷ 30분 = 2포인트
→ 부족해서 선택 불가 (최소 3포인트 필요)
```

### 빙고 완성 시:
```
81칸 × 3포인트 = 243포인트 필요
→ 243 × 30분 = 7,290분
→ 7,290 ÷ 60 = 121.5시간
→ 약 122시간 플레이 필요
```
